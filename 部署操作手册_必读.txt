=============================================================================
                       OpenClaw 云端部署操作指南
=============================================================================

由于安全权限限制，我无法直接连接到您的远程服务器进行操作。
请您按照以下 3 个步骤手动执行，即可完成部署。

-----------------------------------------------------------------------------
步骤 1: 准备工作
-----------------------------------------------------------------------------
1. 确保您拥有一台云服务器 (VPS)。
   - 推荐系统: Ubuntu 22.04 LTS
   - 推荐配置: 2核 CPU, 4GB 内存 (最低 2GB)
2. 确保您知道服务器的 IP 地址 (例如: 1.2.3.4) 和 用户名 (通常是 root)。

-----------------------------------------------------------------------------
步骤 2: 上传部署文件
-----------------------------------------------------------------------------
在您的本地电脑 (当前设备) 打开 PowerShell 或终端，运行以下命令：
(请将 <your-server-ip> 替换为实际的服务器 IP)

scp -r D:\OpenClaw root@<your-server-ip>:~/openclaw-deploy

示例: scp -r D:\OpenClaw root@123.45.67.89:~/openclaw-deploy

-----------------------------------------------------------------------------
步骤 3: 在服务器上执行部署
-----------------------------------------------------------------------------
使用 SSH 登录服务器，并运行我们准备好的自动化脚本：

1. 登录服务器:
   ssh root@<your-server-ip>

2. 进入目录并运行脚本:
   cd ~/openclaw-deploy
   chmod +x deploy_openclaw.sh
   ./deploy_openclaw.sh

脚本将自动完成以下工作：
- 安装 Docker 和环境依赖
- 配置 2GB Swap 交换分区 (防止内存不足)
- 启动 OpenClaw 网关、Redis 和 套利监控插件

-----------------------------------------------------------------------------
步骤 4: 连接使用
-----------------------------------------------------------------------------
部署完成后，OpenClaw 仅监听本地端口 (127.0.0.1:18789) 以确保安全。
您需要通过 SSH 隧道来访问它。

在本地电脑运行:
ssh -L 18789:127.0.0.1:18789 root@<your-server-ip>

然后打开浏览器访问: http://localhost:18789
